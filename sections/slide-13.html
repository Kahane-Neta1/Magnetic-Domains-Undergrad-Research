Domain Disappearance

Graph of the number of polarity reversals per frame as a function of time. The decay constant of the reversal count is noted to be similar across voltage jumps. This constant represents the probability per unit time that a domain will disappear. The probability remains the same across different voltage jumps, meaning the chance of disappearance given that a change occurs is identical. This does not contradict hysteresis width because at lower voltages there is less overall change, so the threshold for change is lower.
